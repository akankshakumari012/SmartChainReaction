# Chain-Reaction-AI

This is a **Chain Reaction game** developed as a hackathon project for **Critical Mass: The Ultimate AI Bot-Making Competition** at **Indian Institute of Technology (IIT), Patna**.  

The game features a **smart AI bot** that plays against you on a 12×8 board. It includes **chain reaction explosions, scoring system, animations**, and a **restart button**.

---

## 🎮 Game Features

- **Player vs AI**: Red (you) vs Green (AI)  
- **12×8 Board**: Standard Chain Reaction layout  
- **Smart AI**: Uses strategy to make intelligent moves  
- **Chain Reactions**: Cells explode when critical mass is reached, spreading to neighbors  
- **1-Second AI Delay**: Fast gameplay, suitable for hackathons  
- **Scoreboard**: Tracks your wins, AI wins, and draws  
- **Restart Button**: Reset the board anytime without losing scores  
- **Winning Animation**: Slash line shows the winning sequence  

---

## 🕹 How to Play

1. Click on an empty cell or a cell you already own.  
2. When a cell reaches its **critical mass**, it **explodes**:
   - **Corner cells**: 2 orbs  
   - **Edge cells**: 3 orbs  
   - **Inner cells**: 4 orbs  
3. Explosions spread to neighboring cells and can cause **chain reactions**.  
4. The goal is to **eliminate all the opponent’s orbs**.
